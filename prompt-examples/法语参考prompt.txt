# 法语 AI 提取 Prompt 参考示例

## 单词提取 Prompt

提取文本中的法语单词，按以下要求生成：

【单词要求】
- 提取常用词汇（A1-B2级别）
- 包含名词、动词、形容词、副词等
- 必须标注阴阳性（le/la）
- 标注音标（IPA格式）

【返回格式】
- type: "word"
- original: 法语单词（含冠词）
- translation: 中文翻译
- wordType: 词性（如：名词阳性、动词、形容词）
- gender: 阴阳性（m./f.，动词等可留空）
- explanation: Markdown格式，包含：
  - 词源分析（拉丁词根等）
  - 词性变化（如动词变位提示）
  - 常见搭配
  - 近义词辨析
- example: 例句（法语+中文翻译）

## 短语提取 Prompt

提取文本中的法语短语和固定搭配：

【短语要求】
- 提取常用短语、习语
- 包含介词搭配
- 标注语体色彩（口语/书面/正式）

【返回格式】
- type: "phrase"
- original: 法语短语
- translation: 中文翻译
- wordType: 短语类型（如：动词短语、介词短语）
- explanation: Markdown格式，包含：
  - 结构分析
  - 使用场景
  - 同义表达
- example: 例句

## 语句提取 Prompt

提取文本中的完整句子：

【语句要求】
- 选择语法完整的句子
- 包含不同的时态和语态
- 优先选择实用场景句

【返回格式】
- type: "sentence"
- original: 法语句子
- translation: 中文翻译
- wordType: 句型（如：陈述句、疑问句、复合句）
- explanation: Markdown格式，包含：
  - 语法结构分析
  - 时态说明
  - 重点词汇解析
