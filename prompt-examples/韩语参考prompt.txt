# 韩语 AI 提取 Prompt 参考示例

## 单词提取 Prompt

提取文本中的韩语单词，按以下要求生成：

【单词要求】
- 提取常用词汇（TOPIK I-III级）
- 区分固有词、汉字词、外来词
- 必须标注词性
- 外来词标注来源语言

【返回格式】
- type: "word"
- original: 韩语单词（含助词）
- translation: 中文翻译
- wordType: 词性（如：名词-汉字词、动词、形容词、副词）
- gender: 留空（韩语无性）
- explanation: Markdown格式，包含：
  - 词源分类（固有词/汉字词/外来词）
  - 汉字词需拆解（如：大統領 → 대통령）
  - 外来词标注原词（如：커피 ← coffee）
  - 语法特征（活用变化提示）
  - 使用场景
- example: 例句（韩语+中文翻译）

## 短语提取 Prompt

提取文本中的韩语短语和惯用表达：

【短语要求】
- 提取常用短语、惯用语
- 包含助词搭配
- 区分敬语和非敬语

【返回格式】
- type: "phrase"
- original: 韩语短语
- translation: 中文翻译
- wordType: 短语类型（如：动词短语、名词短语、惯用语）
- explanation: Markdown格式，包含：
  - 结构分解（助词功能说明）
  - 使用场景
  - 敬语提示
  - 同义表达
- example: 例句

## 语句提取 Prompt

提取文本中的完整句子：

【语句要求】
- 选择语法完整的句子
- 包含不同的时态和句式
- 优先选择实用场景句

【返回格式】
- type: "sentence"
- original: 韩语句子
- translation: 中文翻译
- wordType: 句型（如：陈述句、疑问句、命令句、复合句）
- explanation: Markdown格式，包含：
  - 语法结构分析
  - 助词功能说明
  - 语尾变化解析
  - 重点词汇解析
